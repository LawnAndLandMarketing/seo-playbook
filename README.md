# Lawn & Land Marketing — SEO Playbook

The standardized SEO process we follow for every client, every month. No exceptions.

**Live page:** [lawnlab.dev/seo-playbook](https://lawnlab.dev/seo-playbook)

---

## Philosophy

SEO is a long game. Our job is to build a compounding foundation — technical soundness, local authority, consistent content — that makes our clients the obvious answer when someone searches for lawn care or landscaping in their area.

Every deliverable listed here is performed for every active SEO client unless otherwise documented in their client file.

---

## What We Do

### 1. Google Business Profile (GBP)

**Onboarding (One-Time)**
- Complete profile to 100% — every field filled
- Add all services with descriptions
- Upload photos (exterior, work, team)
- Set service areas accurately
- Verify all business info (hours, phone, website, address)

**Ongoing (Monthly)**
- 52 weekly posts scheduled annually — seasonal, promotional, educational
- Automated review replies via Zapier + Claude AI (responds within minutes)
- GBP optimization audits — categories, attributes, Q&A

---

### 2. Website

**Build (One-Time / Ongoing as Services Expand)**
- Core service pages (e.g., Lawn Care, Landscaping, Hardscaping)
- Sub-service pages for each service (e.g., Mulching, Sod Installation, Retaining Walls)
- Service area pages — city/region-specific landing pages
- All content written for rankability — keyword-targeted, locally relevant

**Ongoing**
- Internal linking between pages as content grows
- Page updates when services or service areas change

---

### 3. Technical SEO

**Onboarding (One-Time)**
- Rank Math setup and configuration on WordPress
- Schema markup — LocalBusiness, Service, FAQPage, BreadcrumbList
- XML sitemap submission to Google Search Console
- robots.txt review and optimization
- 301 redirect audit

**Ongoing**
- Schema updates when services/info changes
- Technical issue resolution (crawl errors, indexing issues surfaced by GSC)

> **Known Gap:** Rank Math configuration is not yet fully standardized across all clients. Standardization is a current priority.

---

### 4. Google Search Console

**Onboarding (One-Time)**
- GSC account setup and verification
- Property configuration (domain vs. URL prefix)
- Sitemap submission

**Ongoing**
- Monthly GSC review — impressions, clicks, average position
- Indexing issue monitoring and resolution
- Identification of keyword opportunities (positions 4–20)

> **Known Gap:** GSC monitoring is not yet on a standardized reporting cadence. This is being formalized.

---

### 5. Content

**Monthly**
- 1 blog post per client — locally relevant, service-targeted, keyword-optimized
- Alt text added to all images on new posts and pages
- Internal links added to and from new content

> **Known Gap:** 1 blog/month is a minimum baseline, not a ceiling. Increasing content cadence is a priority. Alt text on existing pages is an active cleanup effort.

---

### 6. Link Building & Authority

**Onboarding (One-Time)**
- Citation submission via Whitespark — top 50+ directories
- Directory cleanup — NAP consistency audit and corrections

**Monthly**
- 2 press releases via Signal Genesis / Search Atlas
  - Press release #1 links to homepage
  - Press release #2 links to current month's blog post

> **Known Gap:** No ongoing citation building or backlink outreach beyond press releases. Developing a repeatable system for this is a priority.
